# 证明偶数 D(n)=2n：方向 2 与方向 1 的分析

> 数据实证脚本：`verify_dir2.py`（推广 rot4 对角线定律）、`verify_dir1.py`（四染色配额）。
> **实证已于 2026-07-08 完成**（沙箱恢复后运行），下列定理均有数据佐证。

---

## 0. 先校准问题：D(n)=2n 的哪一半是开放的？

- **上界 D(n) ≤ 2n 是平凡的**：每行至多 2 点（否则 3 点同排共线），共 n 行 ⇒ ≤ 2n。
  同理每列至多 2 点。这是 1950 年代就已知的初等式上界。
- **下界 / 存在性 D(n) ≥ 2n（即真能放下 2n 点）才是著名的开放猜想**。
  Heule(2026) 用 SAT 把「存在 2n 解」算到了 n ≤ 72（偶数全为 2n，仅 n=71 例外）。
  对所有偶数 n 的严格数学证明仍然**开放**。

**结论**：所谓「证明偶数 D(n)=2n」，真正的硬骨头是**下界（构造）**，不是上界。
下面两个方向要在这个框架下重新评估。

---

## 方向 2：把「对角线方向互异」推广到所有含 180° 旋转对称的解

### 2.1 推广后的定理（C₂ 定理）

**定理.** 设 S 是偶数 n×n 网格上的一个 no-three-in-line 解（|S|=2n，点互异、无 3 共线），
且 S 关于网格中心 C=((n−1)/2,(n−1)/2) 的 **180° 旋转 R₁₈₀(r,c)=(n−1−r,n−1−c) 不变**。
则：
1. S 可分解为 n 个 **R₁₈₀-轨道**，每个轨道是一对关于 C 中心对称的点 {p, R₁₈₀(p)}；
2. 每个轨道确定一条**过 C 的直线**（即连线 p–R₁₈₀(p)，必过 C）；
3. 这 n 条过 C 的直线**两两不同**（不存在两条轨道落在同一条过 C 的直线上）。

**证明.**
(1) R₁₈₀ 是 2 阶对合，偶数 n 时 C 不是格点，故无不动点，所有轨道 size=2。|S|=2n ⇒ 恰好 n 个轨道。
(2) 轨道 {p,q} 满足 q=R₁₈₀(p)，故 C 是线段 pq 的中点，连线 pq 过 C。
(3) 反证：若两个不同轨道落在同一条过 C 的直线 L 上，则 L 上含有这 4 个点（每轨道 2 点），
其中任取 3 点共线 ⇒ 与「no-three-in-line」矛盾。∎

### 2.2 与 rot4 观察的关系

- rot4 解额外含 90° 旋转，但其 R₁₈₀-轨道仍是 size 2；一个 rot4 轨道（size 4）恰拆分为**2 个 R₁₈₀-轨道**，
  对应正方形的**两条对角线**方向。于是 n 个 R₁₈₀-轨道 = n 条过 C 方向 = 之前观察到的「恰好 n 条互异对角线方向」。
- 因此 rot4 的结论是 C₂ 定理的特例；C₂ 定理把它推广到**所有含半周转对称的类**。

### 2.3 适用范围（群论判定 + 数据实测）

定律**成立当且仅当解的对称群包含关于 C 的半周转 R₁₈₀**。由群结构：

- **含 R₁₈₀ 的类（实证确认 inv180=True）**：
  - **rot2**：对称群 = ⟨R₁₈₀⟩ = C₂；
  - **rot4**：对称群 = C₄，R₁₈₀ = R₉₀²；
  - **dia2**：对称群 = V₄ = {id, R₁₈₀, d₁, d₂}（d₁,d₂ 为两条对角反射）。
    R₁₈₀ = d₁ ∘ d₂，故**dia2 确含 R₁₈₀**（首轮分析误判为「不含」，实测已纠正）。
- **不含 R₁₈₀ 的类（实证确认 inv180=False）**：
  - **iden**：平凡群，无对称；
  - **dia1**：仅含**单条**对角反射 D₁ = {id, d₁}，不含 R₁₈₀；
  - **ort1**（及未采样的 ort2）：正交反射，轴不过 C，不含 R₁₈₀。

这些类的轨道不关于 C 对称，故无「过 C 方向」结构——定律天然不适用。

> **实测佐证（verify_dir2.py，采样 3732 个含 R₁₈₀ 解）**：
> 全部 3732 个解满足「n 条方向两两不平行」，平行对（=4 点共线）出现 **0** 次。
> 其中 inv180=True 的类为 rot2 / rot4 / dia2；inv180=False 的类为 iden / dia1 / ort1。
> 这与上述群论判定完全一致。

### 2.4 它如何服务于「证明 D(n)=2n」

C₂ 定理把「构造一个偶数 n 的 2n 解」微缩成一个组合问题：
> 选 **n** 个「种子点」f₁…f_n，令 S = ⋃ᵢ {fᵢ, R₁₈₀(fᵢ)}（每个种子生成一个对称对，共 n 对 = 2n 点）。
> 要求：(a) n 个对的中点方向两两不同（C₂ 定理已保证过 C 无 3 共线）；
>       (b) 任意**不过 C**的直线不同时穿 ≥3 个 S 中的点（这是真正难的约束）。

换句话说，C₂ 对称性**自动解决了一半的共线风险**（过中心的），把问题压缩到「非中心直线的 3 共线」——
这正是搜索求解器要啃的硬骨头，也是未来可能用数学方法逼近的薄弱点。

## 方向 1：偶数网格四染色配额

### 1.1 每条线至多触及 2 色 ⇒ 上界 D(n) ≤ 2n

按 (r mod 2, c mod 2) 把 n×n 染 4 色（每色 (n/2)×(n/2) 子格）。
任意直线方向 (a,b) 取既约（gcd(a,b)=1），则 gcd 不全为偶，步长改变奇偶性为
(奇,奇)/(奇,偶)/(偶,奇) 三种之一，各使颜色在 **2 色之间交替**。
故**每条直线至多触及 4 色中的 2 色**。

**实测佐证（verify_dir1.py）**：对 |a|,|b|≤6 的所有既约方向扫描，一条线触及的颜色数最大值 = **2**。

由此重证上界：每行 r 的点只落在它的 2 个颜色 (r%2,0)、(r%2,1) 中；
每行至多 2 点 ⇒ 总 ≤ 2n。**但这一步「每行至多 1 点入该色」是错误的**（一行可含同色 2 点），
所以四染色本身并未给出独立于「每行≤2」的新上界——它仍是那个平凡上界，只是换了个视角。

### 1.2 rot4/rct4 解的「完全颜色平衡」（是定理，不是普遍必要条件）

若 S 是 **rot4**（或 rct4）解且 |S|=2n，则 R₉₀（或 D₂）在 4 个颜色类上传递且自由地作用，
故每色类点数必相等 = n/2，且每行恰 2 点。这是一个由对称性保证的**定理**。

**但「每色恰 n/2」绝非任意 2n 解的必要条件。** 实测（verify_dir1.py）：
iden 类 143691 个 2n 解中 55302 个不平衡（如 n=12 计数 `[7,5,5,7]`）；rot2、dia1、dia2 同理。
所以颜色平衡只是 rot4/rct4 的**对称诱导性质**，可作构造过滤器，而非普适定律。

**实测佐证（verify_dir1.py，rot4 类真实 2n 解）**：
对 n=12,14,…,44 的每个 n 采样 4–50 个解，**100% 满足**「每色恰 n/2、每行恰 2」——
这正是上述对称定理的反映，而非「极大解必要性」。

### 1.3 诚实重评估：方向 1 攻不破真正开放的部分

四染色只能给出上界（且等于已知的平凡上界），**不能证明下界/存在性**。
因此「用四染色 + 鸽巢证明 D(n)=2n」这个目标设定本身错位——
鸽巢给出的是 D(n) ≤ 2n，而我们需要的是 D(n) ≥ 2n（构造）。

四染色真正的用处是 **1.2 的平衡条件**：它告诉我们任何 2n 构造必须满足严格的颜色/行平衡，
这可以作为**构造的校验约束**或**排除某些对称类型的判据**，但不是证明本身。

## 附带结构定律：rot4/rct4 解的「距离环人口为 4 的倍数」

对 rot4（及 rct4）解，每个轨道是绕中心 C 的正方形，4 个顶点到 C 等距，故每个轨道给**单一**距离环
贡献恰好 4 个点。因此每个距离环承载的点数必为 **4 的倍数**（= 4 × 同环轨道数）。

> **实测更正（2026-07-08）**：首轮分析笔记称「环人口恒为 4 或 8」系**过小样本**导致的过度归纳。
> 对全部 21601 个 rot4 解（n=12–72）实测，环人口集合 = **{4, 8, 12, 16}**，
> 即同环轨道数可达 1–4，但**始终为 4 的倍数**（0 个反例）。
> 正确且可证的定律是「环人口 ≡ 0 (mod 4)」，比「4 或 8」更强也更干净。
> 推论：rot4 解中每个距离环至少 4 点，故**中心 C 必为某距离环的外接圆心**——
> 这与 rot4 解「含中心作 circumcenter」的观察一致，但这是 C₄ 定理的几何后果，而非缺失中心判据。

## 总体判断（给用户的直接意见）

1. **上界已死**：D(n)≤2n 平凡成立，四染色只是换个证法，不产生新进展。
2. **下界（构造）才是开放核心**，也是本项目最有价值的切入点。
3. **方向 2（对称不变量）远比方向 1 有前途**：
   - C₂ 定理把偶数 2n 解的「过中心共线」风险彻底代数化（方向互异即可）；
   - 真正未决的是「非中心直线的 3 共线」，它正是 rot4/rct4 数据里看似「有机」的部分。
   - 把 C₂ 定理 + 环人口(4 的倍数) + 颜色平衡三者合起来，可能给出**构造偶数 2n 解的组合配方**，
     这比纯鸽巢更接近证明。
4. **现实路线建议**：不要追求「一个定理证遍所有偶数 n」。更可行的论文贡献是
   (a) 把 C₂ 定理及「环人口为 4 的倍数」写成严格引理（本项目独有资产）；
   (b) 用这些不变量**构造/刻画**某一类偶数 n（如 n≡0 mod 4 或素数幂）的 2n 解，
       把「存在性」从计算证据提升为部分数学证明。
