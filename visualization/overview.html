What is a "Missing-Center" Solution?

The grid center \(C = \left(\frac{n-1}{2}, \frac{n-1}{2}\right)\) is the circumcenter of a triple of grid points iff those three points are equidistant from \(C\) — they lie on the same distance ring.

≤2
Points per distance ring
→ center NOT a circumcenter
≥3
Points on same ring
→ center IS a circumcenter

The distance rings are concentric circles centered at the grid center. The squared distance is \(d^2 = (i-c_x)^2 + (j-c_y)^2\). For integer grid coordinates and half-integer centers (even n), all \(d^2\) values are integers — no floating point needed.

Interactive example: n=12 rot4 solution

12×12 grid — distance rings

Distance rings: Each ring contains points at the same squared distance from the grid center. If any ring has ≥3 selected points, the center is a circumcenter.

missing-center = every distance ring has ≤2 points.

The C₄ Theorem — A Proven Result

Theorem. Any No-Three-In-Line solution with C₄ (90° rotational) symmetry must have the grid center as a circumcenter of some triple.

C₄ orbit: 4 points → same distance ring

Why it works:

  1. 90° rotation \(R\) maps a C₄ solution to itself
  2. \(R\) preserves distance from the center
  3. Each orbit has 4 points (even n)
  4. All 4 share the same \(d^2\)
  5. So that ring has ≥4 points
  6. ∴ center is a circumcenter

Corollary: Missing-center solutions cannot have C₄ symmetry. Verified on 34,473 rot4 solutions across 26 even n values.

The Flammenkamp database classifies solutions by their dihedral symmetry. Each class has a different orbit structure, which determines whether missing-center solutions are possible.

iden (no symmetry)

Phase Transitions and Extinction

Three-Phase Evolution of Odd n

Phase 1 — Abundance (n=7–19): Missing-center rate rises to 9.2% at n=13. The iden and rot2 classes coexist.

Phase 2 — Decline (n=21–29): iden disappears; only rot2 survives. Missing-center rate decays asymptotically from 7.8% to 4.8%.

Phase 3 — Extinction (n≥31): rot2 vanishes at n=31 (SAT phase transition). Only rct4 remains, and rct4 always has center as circumcenter. Missing-center solutions are permanently extinct for odd n ≥ 33.

Even n Threshold: n=12

n=6, 8, 10 have zero missing-center solutions. n=12 is the first even n with missing-center solutions (52). The threshold is caused by collinearity constraint density, not ring capacity.
